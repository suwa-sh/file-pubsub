### モデルシステム選定根拠

- 利用者は導入先組織内の運用者・Consumer 担当者に限定され、停止時の影響も組織内のファイル連携遅延にとどまる → 社会的影響が極めて限定的 = **model1**
- 単一インスタンス・HA なしというユーザー確定の設計スコープとも整合
- ただし「データを失わない」(RPO Lv4) と「24h 自動監視」(C.1.1.1 Lv5) は中継基盤としての性格上、model1 デフォルトより高いレベルをユーザー確定で採用
